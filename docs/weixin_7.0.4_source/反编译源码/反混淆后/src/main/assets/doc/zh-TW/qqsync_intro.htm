通訊錄安全助手有什麽作用呢？
1、手機不慎丟失，存儲在手機中的號碼全部丟失，補辦SIM卡容易，但是號碼卻無法找回，好友失去聯系……如果您使用了通訊錄安全助手，可以通過“恢復通訊錄”的功能來解決問題。
2、買了部新手機，滿心歡喜的更換，但是聯系人導出和導入很繁瑣，花費了您大量的時間。可以先在舊手機上使用“備份通訊錄”功能，再到新手機上通過通訊錄安全助手使用“恢復通訊錄”的功能來快速導出手機通訊錄聯系人。

功能操作說明
1、備份：上傳手機通訊錄聯系人到網絡端備份，網絡端原有的數據將被更新。
2、恢復：下載網絡端聯系人到手機端，手機通訊錄原有數據將被更新。

備份的數據保存在哪裏？
1、在通訊錄安全助手裏備份的手機通訊錄數據會保存在QQ同步助手安全中心。
2、數據與您的微信號綁定的QQ號相關聯。如果您不確定您綁定的QQ號是什麽，可以到微信[設置]->[我的賬號]查看當前微信賬號綁定的QQ號。
3、您也可以通過在網頁上QQ同步助手安全中心，登錄您綁定的QQ號來查看到當前備份的數據。

使用通訊錄安全助手備份的信息會不會泄漏？
1、通訊錄安全助手備份用戶的手機通訊錄信息采用加密存儲在服務器，確保了個人數據的安全。
2、在客戶端與服務端通訊時，我們通過數字證書加密通道傳輸信息，保證信息加密傳輸。采用專業的數字證書加密，並對手機客戶端進行了安全認證。
